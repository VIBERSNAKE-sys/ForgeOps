# Article Digest — Your Evidence Base

This file contains detailed proof points from your career, organized by role.
ForgeOps reads this file when generating CVs to find specific metrics,
scope details, and accomplishments to match against JD requirements.

The `validate-cv.mjs` script cross-references every metric in your generated
CV against this file. If a number appears in your CV but not here, it gets
flagged as a potential fabrication.

---

## How to Populate This File

After creating your profile, add verified facts here.
Organize by role, most recent first. Include:

- **Exact metrics** (percentages, counts, dollar amounts, timeframes)
- **Team sizes** and stakeholder counts
- **Tools and methods** used
- **Specific deliverables** created
- **Scope markers** (countries, departments, projects)
